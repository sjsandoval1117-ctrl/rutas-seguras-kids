# Rutas Seguras Kids

## Descripción del proyecto
Aplicación web para la gestión de rutas escolares y estudiantes. Permite crear rutas, asignar estudiantes, visualizar pasajeros por ruta y consultar el clima en tiempo real mediante una API. Incluye validaciones, almacenamiento local (localStorage) y componentes reutilizables con Web Components.

## Instrucciones de ejecución
1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo `index.html` en un navegador web.
4. Ingresar con:
   Usuario: admin  
   Contraseña: 1234
